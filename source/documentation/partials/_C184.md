## C184
```
IF <Follow Up Correlation ID> does not match with <Follow Up Correlation ID> in a request message
     THEN  <National Case Reference Identifier> is "O"
  ELSE IF <Follow Up Correlation ID> matches with <Follow Up Correlation ID> in a request message AND
<National Case Reference Identifier> is present in the request message
  THEN <National Case Reference Identifier> is "R"
    ELSE <National Case Reference Identifier> does not apply.
```
