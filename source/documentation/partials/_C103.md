## C103
```
IF <E-AD/E-SAD Update.Changed Transport Arrangement> is not given
THEN <TRADER New Transport Arranger> does not apply
ELSE IF <E-AD/E-SAD Update.Changed Transport arrangement> is "Consignor" or "Consignee"
THEN <TRADER New Transport Arranger> does not apply
ELSE <TRADER New Transport Arranger> is 'R'.
```
