## C164
```
IF <Destination Type Code> is "Unknown destination" or "Destination - Registered consignee"
THEN <(DELIVERY PLACE) TRADER> does not apply
          <(DELIVERY PLACE) CUSTOMS OFFICE> does not apply
ELSE IF <Destination Type Code> is "Destination - Export"
THEN <(DELIVERY PLACE) TRADER> does not apply
          <(DELIVERY PLACE) CUSTOMS OFFICE> is 'R'
ELSE IF <Destination Type Code> is "Destination - Exempted consignee" or "Destination - Temporary registered consignee"
THEN <(DELIVERY PLACE) TRADER> is 'O'
           <(DELIVERY PLACE) CUSTOMS OFFICE> does not apply
ELSE <(DELIVERY PLACE) TRADER> is 'R'
          <(DELIVERY PLACE) CUSTOMS OFFICE> does not apply
```
