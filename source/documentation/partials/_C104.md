## C104
```
IF <E-AD Split Details.Changed Transport Arrangement> is not given
  THEN <TRADER New Transport Arranger> does not apply
ELSE IF <E-AD Split Details.Changed Transport arrangement> is "Consignor" or "Consignee"
  THEN <TRADER New Transport Arranger> does not apply
  ELSE <TRADER New Transport Arranger> is 'R'.
```
