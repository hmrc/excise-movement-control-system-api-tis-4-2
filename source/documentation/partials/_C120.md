## C120
```
IF <EVENT REPORT.Changed Transport Arrangement> is not given
  THEN <TRADER New Transport Arranger> does not apply
  ELSE IF <EVENT REPORT.Changed Transport arrangement> is "Consignor" or "Consignee"
    THEN <TRADER New Transport Arranger> does not apply
    ELSE <TRADER New Transport Arranger> is 'R'
```
